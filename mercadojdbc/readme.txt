Manual do Usuário para Utilização

Este guia explica como operar a aplicação de gestão de estoque e vendas, destacando as principais características e os passos necessários para executar as operações disponíveis.

Tela Inicial
Ao iniciar a aplicação, você será conduzido à tela inicial, que apresenta um conjunto de abas representando diversas funcionalidades. As abas disponíveis são:

Página de Clientes: Facilita o gerenciamento de clientes.
Página de Registro de Vendas: Possibilita efetuar vendas, incluindo produtos no carrinho.
Página de Finalização de Compras: Apresenta a conclusão da compra, com os produtos selecionados e o total a ser pago.
Página de Estoque: Permite visualizar, adicionar e remover produtos do estoque.
Página de Registro de Vendas
Código de Identificação: Insira o código de identificação do produto desejado.
Incluir Produto: Clique neste botão para adicionar o produto ao carrinho de compras.
Lista de Produtos: Exibe os produtos adicionados ao carrinho, com seus preços.
Excluir Produto: Selecione um produto na lista e clique neste botão para removê-lo do carrinho.
Cliente VIP: Marque esta opção se o cliente for VIP para aplicar desconto.
Pagamento: Clique neste botão para prosseguir para a página de Finalização de Compras.
Página de Finalização de Compras
Lista de Produtos: Exibe os produtos adicionados ao carrinho.
Total: Indica o valor total da compra.
Concluir Compra: Finaliza a compra e exibe uma mensagem de conclusão.
Página de Estoque
Listar Produtos: Clique neste botão para visualizar a lista de produtos em estoque.
Adicionar Produto: Clique neste botão para inserir um novo produto no estoque.
Remover Produto: Clique neste botão para excluir um produto do estoque.
Adicionar Produto ao Estoque
Ao clicar em "Incluir Produto" na página de estoque, um novo diálogo será aberto. Preencha as informações solicitadas (Código de Identificação, Nome, Quantidade, Preço) e clique em "OK" para adicionar o produto. Em caso de erro na entrada de dados, será exibida uma mensagem de erro.

Remover Produto do Estoque
Ao clicar em "Remover Produto" na página de estoque, digite o código de identificação do produto a ser removido. Se o produto existir, ele será retirado do estoque.

Execução Independente da Página de Estoque
Você também pode acessar a página de estoque de forma independente clicando na aba "Página de Estoque" ou utilizando a opção "Abrir Janela de Estoque" no menu. Isso permitirá que você gerencie o estoque em uma janela separada.

Considerações Finais
Este guia oferece uma visão geral das principais funcionalidades da aplicação. Certifique-se de seguir as instruções nas telas para garantir uma experiência adequada. Em caso de dúvidas ou problemas, consulte a documentação ou entre em contato com o suporte técnico.

Use os botões de navegação entre as abas para facilitar a interação com o sistema.
Sempre verifique a precisão dos dados inseridos antes de confirmar o registro ou concluir uma venda.
Para dúvidas ou problemas, consulte a seção de ajuda da aplicação.
